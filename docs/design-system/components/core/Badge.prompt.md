Badge — compact status or metadata label for tags, states, and service indicators.

```jsx
<Badge variant="brand" icon="home">Home visit</Badge>
<Badge variant="accent" pill icon="video">Online now</Badge>
<Badge variant="success">Confirmed</Badge>
<Badge variant="warning">Pending</Badge>
```

Variants: `neutral | brand | accent | success | warning | error | info`. Props: `icon` (leading glyph), `pill` (full radius), `solid` (filled brand). Keep to 1–2 words.
