ReviewItem — a single review on the vet profile: Avatar + Name + Rating + text + date.

```jsx
<ReviewItem author="Camila R." meta="tutora da Mel" rating={5} date="há 2 dias"
  text="A Dra. Helena foi super atenciosa. O atendimento em casa fez toda a diferença." />
```

Stack several in a Card; set `divider={false}` on the last item.
